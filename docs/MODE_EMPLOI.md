# Mode d'emploi DICA Decorator

**Visualisez vos décors DICA sur vos photos en quelques clics.**

---

## En 3 étapes, c'est prêt

```
 1️⃣  Je crée un projet      →    2️⃣  J'ajoute ma photo      →    3️⃣  Je choisis un décor
     (ascenseur, van…)             (JPG, PNG, WEBP)                  → Je télécharge le rendu
```

Temps moyen : **moins de 2 minutes** pour obtenir votre première visualisation.

---

## 1. Se connecter

1. Ouvrez l'application DICA Decorator.
2. Cliquez sur **« Entrer »** depuis la page d'accueil.
3. Saisissez votre **email** et **mot de passe**.
   - Pas encore de compte ? Cliquez sur **« Créer un compte »**.
4. Vous arrivez sur votre **tableau de bord**.

> 💡 Pour vous déconnecter : bouton **« Déconnexion »** en haut à droite.

---

## 2. Créer un projet

1. Sur le tableau de bord, cliquez sur **« + Nouveau Projet »**.
2. Remplissez :

   | Champ | Quoi mettre |
   |-------|-------------|
   | **Titre** | Un nom clair, ex. *« Ascenseur M. Dupont »* |
   | **Cas d'usage** | Ascenseur, Van, Terrasse ou Autre |
   | **Référence client** | Optionnel, votre numéro de dossier |

3. Cliquez sur **« Créer le projet »**.

> ⚠️ Le **cas d'usage** est important : il indique à l'IA quelles surfaces elle peut modifier (murs, parois, sol…) et celles à préserver (plafond, poignées, meubles…).

---

## 3. Ajouter une photo

1. Ouvrez votre projet.
2. Cliquez sur **« Ajouter une photo »**.
3. Sélectionnez une image depuis votre ordinateur.

**Pour un bon résultat :**

| ✅ À faire | ❌ À éviter |
|-----------|-------------|
| Photo nette, bien éclairée | Photo floue ou sombre |
| Vue frontale ou légèrement de biais | Angles extrêmes |
| Résolution ≥ 1024 px | Image très compressée (WhatsApp…) |
| Surfaces bien visibles | Objets qui cachent les murs |

---

## 4. Appliquer un décor

1. Sur votre photo, cliquez sur **« Appliquer un décor »**.
2. Parcourez les catégories : **Métal · Unis · Marbre · Bois · Déco**.
3. Cliquez sur le décor voulu (bordure bleue = sélectionné).
4. Réglez les options si besoin :
   - **Nombre de rendus** : 1 ou 2
   - **Format** : carré, portrait, paysage
   - **Afficher la référence DICA sur l'image** : pratique pour vos présentations client
5. Cliquez sur **« Générer le rendu »**.
6. Patientez **30 à 60 secondes** par rendu ☕.

Les rendus apparaissent automatiquement sous votre photo.

---

## 5. Utiliser vos rendus

Sur chaque rendu, vous disposez de ces actions :

| Icône | Action |
|-------|--------|
| 🔍 | Voir en **plein écran** |
| ❤️ | Ajouter aux **favoris** |
| ⬇️ | **Télécharger** l'image (PNG HD) |
| 🔄 | **Régénérer** une variante |
| 🗑️ | Supprimer le rendu |

> 💡 Les favoris sont sauvegardés sur votre compte et accessibles depuis tous vos appareils.

---

## 6. Bonus : l'Assistant Créatif ✨

Pour aller plus loin, cliquez sur **« Libérez votre imagination »** depuis le tableau de bord.

Vous pouvez lui demander, en langage naturel :

```
« Crée un mood board salle de bain avec nos décors marbre »
« Imagine un ascenseur luxueux avec du bois chaleureux »
« Combine ce van avec le décor Inox Brossé »
```

Vous pouvez même **uploader jusqu'à 5 images** (le lieu, un décor, des personnes…) et demander à l'IA de les **combiner** en une seule scène.

---

## Astuces express

- **Vous n'aimez pas un rendu ?** Cliquez sur 🔄 pour en obtenir une autre variante, sans changer de décor.
- **Pour vos présentations client**, activez l'option **« Afficher les références DICA »** : le code du décor apparaîtra directement sur l'image.
- **Mode nuit** 🌙 : cliquez sur l'icône ☀️/🌙 dans l'en-tête pour basculer.
- **Quota quotidien** : 50 rendus par jour, remis à zéro à minuit.

---

## En cas de pépin

| Problème | Que faire |
|----------|-----------|
| Le rendu ne correspond pas à mes attentes | Vérifiez le **cas d'usage**, puis régénérez (🔄) |
| La génération échoue | Rafraîchissez la page et réessayez |
| Photo refusée | Vérifiez le format (JPG/PNG/WEBP) et la taille |
| Mot de passe oublié | Cliquez sur **« Mot de passe oublié »** sur la page de connexion |
| Besoin d'aide | 📧 **support@dica-france.com** |
